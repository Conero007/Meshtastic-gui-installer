# Meshtastic-gui-installer

Cross Platform GUI for installing Meshtastic Firmware, aggressively alpha software.

# install

```
python3 -m venv venv
venv/bin/activate
pip install -r requirements.txt
```

# running

* ensure the file is executable:

```
chmod +x installer.py
```

* run it
```
./installer.py
```
